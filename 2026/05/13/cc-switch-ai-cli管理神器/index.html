CC Switch —— 一站管理 Claude Code / Codex / Gemini CLI 的瑞士军刀

发表于2026-05-13|更新于2026-05-13|教程
|总字数:1.3k|阅读时长:4分钟|浏览量:

如果你同时使用 Claude Code、Codex、Gemini CLI 等多个 AI 编程 CLI 工具，你一定经历过这些痛苦：改配置文件切来切去、API Key 记不住、供应商地址找半天、MCP 服务器每个工具配一遍……

CC Switch 就是来解决这些问题的。它由 GitHub 用户 farion1231 开发，开源免费（MIT 协议），截止 2026 年初已斩获 5.4 万 Star。

一、CC Switch 是什么？

CC 是一款基于 **Tauri 2** 构建的跨平台桌面应用，核心定位是 **AI CLI 工具的全能配置管理助手**。 简单说：它是一个可视化的配置面板，让你不用再手动编辑 JSON / TOML / .env 文件来切换 AI 模型供应商。

支持的应用：Claude Code、Codex（OpenAI）、Gemini CLI、OpenCode、OpenClaw、Hermes Agent

二、核心功能一览

1. 多供应商一键切换

内置 50+ 预设供应商，覆盖几乎所有主流 AI API 平台：

分类 供应商
官方 Anthropic、OpenAI、Google Gemini
国产 DeepSeek、Kimi、智谱 GLM、Moonshot、百川、Qwen
聚合平台 OpenRouter、SiliconFlow、AIHubMix、NVIDIA NIM
云服务 AWS Bedrock、Azure、Google Cloud

支持 Anthropic Messages 原生格式和 OpenAI Chat Completions 兼容格式，无论供应商用哪种 API 格式都能适配。

切换供应商只需在 CC Switch 中选择对应配置，所有关联的 CLI 工具即时生效，无需重启终端。

2. MCP 服务器统一管理

单一面板管理所有应用的 MCP 配置

支持 stdio / http / sse 三种传输类型

内置预设模板（fetch、context7、sequential-thinking 等）

冲突检测与字段校验

跨 Claude Code / Codex / Gemini CLI / OpenCode 同步

不用再每个工具重复配置一遍 MCP 了，CC Switch 自动帮你做双向同步。

3. Skills 管理

这是让我最惊喜的功能之一。CC Switch 可以：

自动扫描 GitHub 上的热门 Claude Skills 仓库

一键安装 / 卸载 / 更新 Skills，支持批量操作

支持从本地 ZIP 安装

内置 baoyu-skills 等预设仓库

版本管理和回滚

相当于一个 Skills 的应用商店，不用手动下载解压拷贝了。

4. 本地路由 & 故障转移

内置了一个高性能 HTTP 本地代理，提供：

能力 说明
请求监控 实时查看 API 请求/响应
自动故障转移 主供应商挂掉自动切备用
熔断器 连续失败达阈值自动暂停
健康监控 供应商延迟可视化面板
用量统计 Token 消耗、成本追踪、缓存命中率

还有一个Claude Rectifier功能——修复第三方 API 网关的 thinking block 签名格式问题，让 DeepSeek 等国产模型在 Claude Code 中也能完整输出思考过程。

5. 更多实用功能

功能 简述
系统托盘 托盘秒切供应商，不用打开主窗口
会话管理 浏览、搜索、恢复、删除跨应用对话历史
云同步 WebDAV / Dropbox / OneDrive / iCloud 多设备同步配置
深链接 ccswitch:// 协议一键导入供应商/MCP 配置
Prompts Markdown 实时编辑预览，跨应用同步 CLAUDE.md
三语界面 中文 / English / 日本語

Windows

双击安装即可，支持 Windows 10+。

Linux

四、实际使用体验

我把 CC Switch 用在日常开发中一段时间，最大的感受是：

配置效率

以前换一个供应商要改 Claude Code 的 settings.json，再改 Codex 的 TOML，还要改 .env……现在在 CC Switch 里选一下，所有工具自动生效。

MCP 不再重复配置

MCP 服务器在 CC Switch 里配置一次，Claude Code 和 Codex 都能用，新增工具时也只需要加一次。

Skills 安装超方便

之前装 Skills 要手动 clone 或者下载 ZIP 解压到 .claude/skills/ 目录，现在在 CC Switch 里浏览 → 点击安装 → 搞定。

故障转移保住效率

用 DeepSeek 的时候偶尔会遇到 API 限流，本地路由自动切到备用供应商，基本无感切换。

五、技术架构

CC Switch 的技术选型也值得一看：

前端： React 18 + TypeScript + Vite + TailwindCSS + shadcn/ui + TanStack Query v5
后端： Rust（Tauri 2）
数据存储： SQLite
跨平台： Windows / macOS / Linux（x64 + arm64）

Tauri 2 让它体积小巧（相比 Electron 应用），同时 Rust 保证了本地代理的性能。

六、总结

推荐指数：满分

如果你每天和 Claude Code / Codex / Gemini CLI 打交道，CC Switch 是必装工具。它把”改配置”这件事从 5 分钟压缩到 5 秒，加上 MCP 和 Skills 管理、本地路由这些锦上添花的功能，完全可以取代手动管理配置文件的原始方式。

项目地址：https://github.com/farion1231/cc-switch
