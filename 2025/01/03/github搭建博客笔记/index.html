任务1. 关于搭建github博客的步骤学习笔记

（所用设备：拯救者y9000p）

搭建流程总览

准备工作——注册github

首先，打开 github.com，来到github的主界面（如下）

GitHub首页

点击右上角进行注册（使用邮箱注册），登录。

STEP1 — 进行 Fork（重要）

打开 mzlogin.github.io，进入以下页面：

博客模板主页

上图即为博客的模版，往下拉：

接下来开始进行 Fork。

往下拉页面会有 Fork 指南，下面讲述详细步骤。

点击 Fork 选项，然后 Create a new fork。由于个人账号已经完成，以下部分图片来自王鑫学长教学截取。

Description 描述自己博客，由自己撰写，可选可不选。

进行设置 Settings：

Fork 后在浏览器网址搜索自己的网站（如上图：ZenQG.github.io），进入以下界面，加载不出来需等几分钟。

博客首次加载成功

当出现这个界面，证明前面的步骤已经成功，第一步完成！
