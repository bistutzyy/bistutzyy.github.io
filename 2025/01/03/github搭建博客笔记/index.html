Welcome to my first passage!

任务1.关于搭建github博客的步骤学习笔记?

(所用设备：拯救者y9000p)

部分网址：github.com；analytics.google.com;mzlogin.github.io；https://giscus.app/zh-CN?

准备工作——注册github

首先，打开第一个网址github.com，来到github的主界面（如下）

点击右上角进行注册（使用邮箱注册），登录

STEP1——进行Fork（重要）

打开第三个网址mzlogin.github.io，进入以下页面：

上图即为博客的模版，往下拉

接下来开始进行fork

注：往下拉会有fork指南，接下来讲述详细步骤

点击fork选项，然后create a new fork，由于个人账号已经完成，以下部分图片来自王鑫学长教学截取

Description描述自己博客，由自己撰写，可选可不选

进行设置settings

fork后在浏览器网址搜索自己的网站（如上图：ZenQG.github.io），进入以下界面，加载不出来需等几分钟

当出现这个界面，证明前面的步骤已经成功，第一步完成

STEP2——处理细节，装饰博客

第一步：删除CNAME文件，此文件与域名有关，有需要可自行修改

第二步——修改配置——装饰自己博客

首先，打开_config.yml,进入编辑

然后，根据自己的要求更改一些内容

接着是修改评论方面，打开https://giscus.app/zh-CN

完成相应要求

确保自己的账号是公开的

打开settings，在打开后下拉找到discussions勾选，勾选后出现绿色的set up discussions进行点击，账号已完成，故省略

discussions打开成功，然后安装giscus这个app

打开后进行下载和配置，完成后来到以下界面

重新回到https://giscus.app/zh-CN进行仓库验证

验证成功后进入discussion分类

然后启用giscus，首先进行复制（由于不兼容可不复制后面比对修改）

接下来是进行谷歌数据分析，打开第二个网址analytics.google.com，没有账号需自己注册+科学上网工具

谷歌分析展示的页面由于为手机操作效果不好，建议使用电脑，展示如下

将自己的id输入后完成配置操作

附4.关于谷歌账号的注册

谷歌账号注册需要科学上网工具，正常注册是需邮箱和国外手机号
